# Chiaki Web Overlay

This source contains reviewed unigram additions derived from Traditional Chinese
Wikipedia and Chiaki's own social media posts. Only derived lexicon rows are
redistributed here:

```text
qstring<TAB>phrase<TAB>weight<TAB>tags
```

The release builder imports `explicit.tsv` after the project modern overlay and
before variant demotion policy.

## Files

- `explicit.tsv`: reviewed explicit-qstring unigram additions.
- `source-inventory.sha256`: generated by `cargo run --release -- prepare-release`.
